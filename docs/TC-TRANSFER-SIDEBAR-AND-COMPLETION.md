## 5. E2E 자동화 반영 사항

- **기본 3개(병합/압축/변환)**  
  현재 페이지와 관계없이, 해당 이름의 버튼이 보일 때만 클릭 가능.  
  다른 페이지(편집·회전 등)로 이동할 때는 **먼저 "더 살펴보기" 클릭 후** 해당 버튼 클릭해야 함.
- **완료 카드**  
  "완료" 문구, 체크 아이콘, 파일명·크기, 다운로드·다시 편집하기는 역할/이름으로 셀렉터 지정 가능.

### 반영된 E2E 스펙

- `web/e2e/ui-buttons.spec.ts`  
  - 전달 버튼 클릭 시: `inMoreMenu: true` 인 항목(편집, 회전, 이미지 PDF 변환, 페이지 번호, 블라인드)은 "더 살펴보기" 클릭 후 해당 버튼 클릭.  
  - 추가: 완료 카드 "완료" 문구·다운로드 표시 검증, "더 살펴보기" 클릭 시 PDF 편집 버튼 노출 검증.
- `web/e2e/transfer-persistence.spec.ts`  
  - 회전→편집, 편집→회전, 페이지 번호→회전 등 "더 살펴보기" 안의 버튼으로 이동하는 시나리오에 "더 살펴보기" 클릭 단계 추가.
